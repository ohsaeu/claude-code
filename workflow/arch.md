## > 온라인 쇼핑몰 서비스를 위한 마이크로서비스 아키텍처를 설계해줘.                                                       
- 주요기능: 사용자 인증, 상품관리, 주문 처리, 결제, 배송 추적

## > 다음 기술 스택의 호환성을 검증해줘

- Backend: NestJs + TypeScript
- Database: Postgresql + redis
- DevOps: Docker + Kubernetes
