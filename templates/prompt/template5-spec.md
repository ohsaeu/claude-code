# 기술 스택 선정 프롬프트
> 앞서 기획한 프로젝트에 적합한 기술 스택을 추천해줘.

고려사항:
- 개발 속도 (빠른 프로토타이핑)
- 클라이언트 요구사항
- 배포 및 유지보수
- 비용 효율성

비교 기준:
- 프론트 엔드: React vs Vue vs Angular
- 백엔드: Node.js vs Python vs Java
- 데이터베이스: PostgreSQL vs MongoDB vs Firebase
- 배포: AWS vs Netlify vs Vercel
- 상태관리: Redux vs Zustand vs COntext API

결과물:
- 추천 기술 스택 (이유 포함)
- 단계별 가이드
- 대안 기술 스택
- 마이그레이션 계획
